## Three minutes to a usable database

```bash
git clone https://github.com/web-casa/supalite.git
cd supalite
./setup.sh
```

That's it. `./setup.sh` generates all secrets, starts the stack, and prints your admin token. Open [http://localhost:8000/admin/](http://localhost:8000/admin/).
