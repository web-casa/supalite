## 三分钟搞出可用数据库

```bash
git clone https://github.com/web-casa/supalite.git
cd supalite
./setup.sh
```

就这样。`./setup.sh` 自动生成所有密钥、启动栈、打印管理令牌。打开 [http://localhost:8000/admin/](http://localhost:8000/admin/)。

## 为什么用 SupaLite？

<CardGrid stagger>
  <Card title="一键启动" icon="rocket">
    `./setup.sh` 自动生成密钥、启动整套服务、打印管理令牌。从 `git clone` 到可用数据库三分钟搞定。
  </Card>
  <Card title="只装必需的" icon="puzzle">
    Postgres + PostgREST + GoTrue + Studio。砍掉 Storage、Edge Functions、Realtime——大多数小型部署用不上。
  </Card>
  <Card title="开箱即用的运维面板" icon="setting">
    浏览器里就能做备份、密钥轮换、实时日志、会话检查、数据库维护。不用钻 `kubectl exec` 的兔子洞。
  </Card>
  <Card title="独立开发者友好" icon="heart">
    多前端 CORS、定时备份+保留策略、Let's Encrypt 一键 HTTPS。给单人单 VPS 设计。
  </Card>
